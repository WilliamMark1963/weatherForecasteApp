### Weather Forecaste App
This Webpage is built by K V William Mark. 

## Core Functionality
User can search the weather conditions in a particular city and he/she can also get there current location weather details of the current day. 
The webpage will also display the next 5 days weather considering current day weather report in it.
User can change the reading from (default) celsius to fahrenhite for current day.
All searched cities are stored in local storage and served as search history.
